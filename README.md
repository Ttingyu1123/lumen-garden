# ✨ 星光庭園 Lumen Garden

**🎮 線上遊玩：<https://game-zombie.tingyudeco.com/>**
**📜 攻略說明書：<https://game-zombie.tingyudeco.com/manual.html>**
（備用網址：<https://ttingyu1123.github.io/lumen-garden/>）

一款玩法靈感來自《植物大戰殭屍》的**原創** 2D 橫向塔防遊戲。
所有角色、命名與美術均為原創，未使用任何官方素材。

純 HTML + CSS + JavaScript，零依賴、零建置步驟。美術用 emoji（辨識度實測
優於自繪向量圖），音效為 Web Audio 即時合成 — 整個專案零外部素材檔。

## 🚀 執行方式

**直接雙擊 `index.html`** 即可在瀏覽器遊玩（刻意不用 ES modules，`file://` 協議直接可跑）。

或起本地伺服器（任選其一）：

```bash
# Python
python -m http.server 8000
# Node
npx serve .
```

然後開 `http://localhost:8000`。

## 🎮 玩法

- 🏆 **戰役模式**：5 個規則各異的關卡（經典 / 無天降光珠 / 巨石障礙 / 快速敵混編 / 雙 Boss），各擋 10 波即勝，依剩餘炸彈評 1~3 星
- ♾️ **無盡模式**：第 11 波起公式生波，**每 5 波增益三選一**（Roguelike，可疊加），拚最高波次，紀錄存 localStorage
- 開場 **15 秒準備時間**（畫面上方倒數），先鋪經濟
- 點上方**卡牌**選單位 → 點**草地格子**放置（再點一次卡牌可取消選取）
- ⬆️ **升級**：沒選卡牌時點場上單位，最高 Lv3（血量/傷害 ×1.5 每級、生產加速）
- 🥄 **鏟子**：移除單位並退回總投資（含升級費）一半
- 🌟 **星光炸彈**：每列一顆最後防線，敵人破防時自動引爆清光該列，僅此一次
- 點擊場上的 ✨ **光珠**收集「光能」（定時天降；聚光菇也會生產）
- 波次越後面數量越多、血量越厚（每波 +6%，無盡模式無上限）
- 🏅 **8 項成就** + 關卡星級，皆存 localStorage
- 📱 **PWA**：手機可「加入主畫面」安裝，離線可玩
- `P` 鍵或右上角 ⏸ 暫停；🔊 切換音效；❓ 開遊戲說明

## 📁 檔案結構

```
├── index.html            # 入口：HUD、卡牌列、canvas、畫面 overlay、遊戲說明、PWA meta
├── manifest.webmanifest  # PWA 安裝資訊
├── sw.js                 # Service Worker（stale-while-revalidate 離線快取）
├── icons/                # PWA 圖示（192/512/apple-touch，canvas 生成）
├── css/style.css         # 全部樣式
└── js/                   # 依 <script> 順序載入（前面定義全域，後面使用）
    ├── config.js         # 全部數值資料表：單位/敵人/波次/關卡/難度曲線（調平衡只改這檔）
    ├── state.js          # 全域狀態 G + resetGame()
    ├── progress.js       # 關卡星級與成就（localStorage）
    ├── sfx.js            # Web Audio 合成音效 + 開關
    ├── grid.js           # 5×9 格子與座標換算
    ├── resources.js      # 光能與光珠（生成/下落/點擊收集/過期）
    ├── perks.js          # 無盡模式 Roguelike 增益（三選一資料表 + 抽選）
    ├── projectiles.js    # 投射物飛行與命中（減速 / 濺射 / 貫穿）
    ├── units.js          # 單位放置/升級、生產/射擊/地雷行為、受擊死亡
    ├── enemies.js        # 敵人移動、護甲、減速、飛行、分裂、Boss 召喚/碎石
    ├── waves.js          # 波次機、關卡規則套用、無盡公式生波
    ├── ui.js             # 卡牌列、選關/成就/增益畫面、toast、畫面切換
    ├── render.js         # canvas 渲染層（等級星星、飛行浮動、地雷警戒、巨石格）
    └── main.js           # 遊戲迴圈、輸入（滑鼠+觸控）、SW 註冊
```
