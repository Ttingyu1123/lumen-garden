# ✨ 星光庭園 Lumen Garden

**🎮 線上遊玩：<https://game-zombie.tingyudeco.com/>**
**📜 攻略說明書：<https://game-zombie.tingyudeco.com/manual.html>**
（備用網址：<https://ttingyu1123.github.io/lumen-garden/>）

一款玩法靈感來自《植物大戰殭屍》的**原創** 2D 橫向塔防遊戲。
所有角色、命名與美術均為原創，未使用任何官方素材。

純 HTML + CSS + JavaScript，零依賴、零建置步驟。美術用 emoji（辨識度實測
優於自繪向量圖），音效為 Web Audio 即時合成 — 整個專案零外部素材檔。

## 🚀 執行方式

**直接雙擊 `index.html`** 即可在瀏覽器遊玩（刻意不用 ES modules，`file://` 協議直接可跑）。

或起本地伺服器（任選其一）：

```bash
# Python
python -m http.server 8000
# Node
npx serve .
```

然後開 `http://localhost:8000`。

## 🎮 玩法

- 兩種模式：🏆 **戰役**（擋下 10 波即勝利）、♾️ **無盡**（第 11 波起公式生波，拚最高波次，紀錄存 localStorage）
- 開場 **15 秒準備時間**（畫面上方倒數），先鋪經濟
- 點上方**卡牌**選單位 → 點**草地格子**放置（再點一次卡牌可取消選取）
- 🥄 **鏟子**：移除單位並退回一半光能
- 🌟 **星光炸彈**：每列一顆最後防線，敵人破防時自動引爆清光該列，僅此一次
- 點擊場上的 ✨ **光珠**收集「光能」（定時天降；聚光菇也會生產）
- 波次越後面數量越多、血量越厚（每波 +6%，無盡模式無上限）
- `P` 鍵或右上角 ⏸ 暫停；🔊 切換音效；❓ 開遊戲說明

| 單位 | 成本 | 冷卻 | 功能 |
|------|------|------|------|
| 🍄 聚光菇 | 50 | 7s | 每 8 秒生產一顆光珠（+25 光能） |
| 🌵 棘刺射手 | 100 | 7s | 同列有敵人時發射棘刺（20 傷害） |
| ❄️ 寒霜蓮 | 75 | 9s | 霜彈傷害低（10），但使敵人減速 50%（3 秒） |
| 🌺 爆炎花 | 150 | 10s | 火彈爆裂（26），濺射同列附近敵人（13） |
| 🪨 磐石守衛 | 50 | 15s | 500 血高血量肉盾，純擋路 |

| 敵人 | 血量 | 特性 |
|------|------|------|
| 🧟 暗影慢行者 | 200 | 慢速主力，遇單位停下啃咬 |
| 👺 暗影疾走者 | 130 | 快速型，血少跑快 — 用寒霜蓮剋制 |
| 👹 暗影重甲兵 | 180 + 🛡️150 | 護甲先吸收傷害，打破前不掉血 |
| 😈 暗影君王（Boss） | 1800 + 🛡️400 | 巨型頭目：戰役最終波壓軸、無盡每 5 波現身，巨體碰撞框更大 |

## 📁 檔案結構

```
├── index.html          # 入口：HUD、卡牌列、canvas、畫面 overlay、遊戲說明
├── css/style.css       # 全部樣式
└── js/                 # 依 <script> 順序載入（前面定義全域，後面使用）
    ├── config.js       # 全部數值資料表：單位/敵人/10 波波次/難度曲線（調平衡只改這檔）
    ├── state.js        # 全域狀態 G + resetGame()
    ├── sfx.js          # Web Audio 合成音效 + 開關
    ├── grid.js         # 5×9 格子與座標換算
    ├── resources.js    # 光能與光珠（生成/下落/點擊收集/過期）
    ├── projectiles.js  # 投射物飛行與命中（含減速 payload）
    ├── units.js        # 單位放置檢查、生產/射擊行為、受擊死亡
    ├── enemies.js      # 敵人移動、護甲、減速、擋路偵測、攻擊、破防判定
    ├── waves.js        # 10 波波次機、準備/波間倒數、勝利判定
    ├── ui.js           # 卡牌列、冷卻遮罩、暫停/音效/說明按鈕、toast、畫面切換
    ├── render.js       # canvas 渲染層（動畫：受傷閃白、放置彈出、結冰、尾焰）
    └── main.js         # 遊戲迴圈、輸入、暫停控制
```

## 🔧 後續可擴充方向

1. **新單位**：在 `config.js` 的 `UNIT_TYPES` 加一筆即自動出現在卡牌列與說明表；新行為型別（如範圍攻擊 `splash`）在 `units.js` 加一個分支
2. **新敵人**：`ENEMY_TYPES` 加資料 + `WAVES` 排進波次即可（如會跳過守衛的飛行型、死後分裂型）
3. **關卡系統**：把 `WAVES` 抽成多關卡陣列，加關卡選擇畫面
4. **音效升級**：`sfx.js` 的 `play()` 換成真音檔（Howler.js 或原生 Audio），介面不用動
5. **美術升級**：委託美術做 sprite sheet（自繪 SVG 已試過，小尺寸辨識度不如 emoji，見 git 歷史 955ab6d）
6. **進階機制**：單位升級、Boss 特殊技（召喚小兵、踩碎單位）、多樣 Boss
7. **關卡系統**：把 `WAVES` 抽成多關卡陣列，加關卡選擇畫面
8. **成就系統**：`localStorage` 記錄勝場、零炸彈通關等成就
