# ✨ 星光庭園 Lumen Garden

一款玩法靈感來自《植物大戰殭屍》的**原創** 2D 橫向塔防遊戲。
所有角色、命名與美術均為原創（emoji placeholder），未使用任何官方素材。

純 HTML + CSS + JavaScript，零依賴、零建置步驟。

## 🚀 執行方式

**直接雙擊 `index.html`** 即可在瀏覽器遊玩（刻意不用 ES modules，`file://` 協議直接可跑）。

或起本地伺服器（任選其一）：

```bash
# Python
python -m http.server 8000
# Node
npx serve .
```

然後開 `http://localhost:8000`。

## 🎮 玩法

- 點上方**卡牌**選單位 → 點**草地格子**放置（再點一次卡牌可取消選取）
- 點擊場上的 ✨ **光珠**收集「光能」（會定時天降；聚光菇也會生產）
- 擋下 **5 波**暗影族；任一敵人走到最左邊即失敗

| 單位 | 成本 | 冷卻 | 功能 |
|------|------|------|------|
| 🍄 聚光菇 | 50 | 7s | 每 8 秒生產一顆光珠（+25 光能） |
| 🌵 棘刺射手 | 100 | 7s | 同列有敵人時發射棘刺（20 傷害） |
| 🪨 磐石守衛 | 50 | 15s | 500 血高血量肉盾，純擋路 |

| 敵人 | 血量 | 特性 |
|------|------|------|
| 🧟 暗影慢行者 | 200 | 慢速，遇單位停下啃咬 |
| 👺 暗影疾走者 | 130 | 快速型，血少跑快 |

## 📁 檔案結構

```
├── index.html          # 入口：HUD、卡牌列、canvas、三個畫面 overlay
├── css/style.css       # 全部樣式
└── js/                 # 依 <script> 順序載入（前面定義全域，後面使用）
    ├── config.js       # 全部數值資料表（調平衡只改這檔）
    ├── state.js        # 全域狀態 G + resetGame()
    ├── grid.js         # 5×9 格子與座標換算
    ├── resources.js    # 光能與光珠（生成/下落/點擊收集/過期）
    ├── projectiles.js  # 投射物飛行與命中
    ├── units.js        # 單位放置檢查、生產/射擊行為、受擊死亡
    ├── enemies.js      # 敵人移動、擋路偵測、攻擊、破防判定
    ├── waves.js        # 5 波波次機與勝利判定
    ├── ui.js           # 卡牌列、冷卻遮罩、toast、橫幅、畫面切換
    └── main.js         # 遊戲迴圈、輸入、canvas 渲染
```

## 🔧 後續可擴充方向

1. **新單位**：在 `config.js` 的 `UNIT_TYPES` 加一筆即自動出現在卡牌列；新行為型別（如範圍攻擊 `splash`）在 `units.js` 加一個分支
2. **新敵人**：`ENEMY_TYPES` 加資料 + `WAVES` 排進波次即可（如戴盔高血量型、會跳過守衛的飛行型）
3. **關卡系統**：把 `WAVES` 抽成多關卡陣列，加關卡選擇畫面
4. **音效**：Web Audio API 產生合成音效（射擊/放置/收集），維持零素材依賴
5. **美術升級**：emoji 換成 SVG sprite 或 canvas 繪製的向量角色；加入單位放置/死亡動畫
6. **進階機制**：鏟子（移除單位）、割草機式最後防線、夜間特殊資源、Boss 波
7. **存檔**：`localStorage` 記錄最高波次與勝場
8. **行動裝置**：加 touch 事件與 responsive canvas 縮放
